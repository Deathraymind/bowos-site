Hyprland Keybinds

Window Management

SUPER + Q Kill active window
SUPER + DELETE Exit Hyprland
SUPER + W Toggle floating window
SUPER + A Open Rofi (drun)
SUPER + G Toggle group (dwindle)
ALT + RETURN Toggle fullscreen

Screenshots

SUPER + P Take a screenshot (Grim + Swappy)

Application Shortcuts

SUPER + T Open Kitty terminal
SUPER + E Open Nautilus (file manager)
SUPER + C Open Visual Studio Code
SUPER + F Open Zen Browser

Workspace Management

SUPER + [1-0] Switch to workspace 1-10
SUPER + SHIFT + [1-0] Move window to workspace 1-10
SUPER + CTRL + [←→] Switch workspaces relative to active
SUPER + CTRL + ALT + [←→] Move window to relative workspace

Audio Controls

XF86AudioPlay/Pause Play/Pause media
XF86AudioNext Next track
XF86AudioPrev Previous track
XF86AudioRaiseVolume Increase volume
XF86AudioLowerVolume Decrease volume
XF86AudioMute Toggle mute

Brightness Controls

XF86MonBrightnessUp Increase brightness
XF86MonBrightnessDown Decrease brightness
